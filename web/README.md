# 物业管理系统前端

物业管理系统前端，面向物业管理业务的 Web 管理界面，支持业主、房产、费用、报修、公告等日常操作。

## 项目简介

本项目基于 Vue 3、Vite、TypeScript 和 Element Plus 开发，用于对接物业管理系统后端接口，提供物业管理人员和相关角色使用的后台管理界面。

## 主要功能

- 登录认证、角色权限和动态菜单
- 小区、楼栋、单元、房屋和业主管理
- 车辆、车位、停车和物业费用管理
- 报修、投诉、访客和公告管理
- 消息中心、客服会话和站内沟通
- 个人资料、文件上传和日志查看

## 本地运行

```sh
npm install
```

### 启动开发环境

```sh
npm run dev
```

### 类型检查并构建生产版本

```sh
npm run build
```

### 代码检查

```sh

```
